# happy-teeth-backend
Backend for the Happy Teeth web application.

## Tips
The tips that were written on the code were personally written by me (Donovan Thach), and the sources of where the tips come from are provided in the sources below:

**Caries:** https://www.nidcr.nih.gov/health-info/tooth-decay/more-info/tooth-decay-process (NIH)
**Gingivitis:** https://www.mayoclinic.org/diseases-conditions/gingivitis/diagnosis-treatment/drc-20354459 (Mayo Clinic)
**Tooth Discoloration:** https://my.clevelandclinic.org/health/symptoms/10958-tooth-discoloration (Cleveland Clinic)
**Ulcer/Lesion/Xerostomia:** https://my.clevelandclinic.org/health/diseases/21766-mouth-ulcer and https://www.mayoclinic.org/diseases-conditions/dry-mouth/expert-answers/dry-mouth/faq-20058424 (Cleveland/Mayo Clinic)
**Calculus:** https://my.clevelandclinic.org/health/diseases/25102-tartar (Cleveland Clinic)
**Plaque:** https://my.clevelandclinic.org/health/diseases/10953-plaque (Cleveland Clinic)

**Note:** There is a disclaimer for tips, this model will never be 100% accurate, and seeking professional help is still strongly advised if possible.
